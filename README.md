# RL-Mapless-Navigation
Pytorch implementation of several RL algorithms for mapless navigation.

## Requirement
- python 3.6
- pytorch >= 1.0.0
- openCV
- numpy

## Training
```
# Deep Deterministic Policy Gradient (DDPG)
python main_ddpg.py

# Soft Actor-Critic (SAC)
python main_sac.py
```

## Evaluation
```
python rl_eval.py
```
